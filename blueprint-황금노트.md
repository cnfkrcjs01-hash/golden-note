# 황금노트 웹사이트 시스템 설계서

> 작성일: 2026-03-17
> 목적: 개인 업무효율 향상을 위한 로컬 웹 애플리케이션

---

## 1. 작업 컨텍스트

### 배경 및 목적
매년 3월 피복비 지급 시 수백 명의 직원 데이터를 수동으로 검토하여 대상자를 선별하는 반복 업무를 자동화한다.
향후 메뉴를 확장하여 다양한 업무 자동화 기능을 추가할 수 있는 플랫폼으로 설계한다.

### 범위
- 포함: 피복비 대상자 자동 선별 (메뉴 1)
- 제외: 서버 배포, 사용자 인증, DB 저장

### 입출력 정의

| 항목 | 내용 |
|------|------|
| **입력** | 직원 데이터 Excel 파일 (.xlsx) 업로드 |
| **출력** | 화면 결과 표시 + 판정 완료 Excel 다운로드 |
| **트리거** | 사용자가 파일 업로드 후 "판정 실행" 클릭 |

### 제약조건
- 로컬 HTML 파일로 실행 (서버 불필요)
- 모든 처리는 클라이언트 사이드 JavaScript
- SheetJS(xlsx) 라이브러리로 Excel 읽기/쓰기

## 2. 피복비 지급 기준 (2025년 기준)

### 기준년도 설정
- 매년 3월 지급, 기준년도만 변경 (날짜 로직은 고정)
- 기준년도: 사용자가 웹에서 선택 가능

### 5가지 제외 규칙

| # | 규칙명 | 기준 | 로직 |
|---|--------|------|------|
| ① | 3개월 미만 근로자 | 입사일 기준 | {기준년도}.12.01 이후 입사 → 제외 |
| ② | 퇴사자/퇴사예정자 | 퇴사일 기준 | {기준년도}.03.19 이전 퇴사 → 제외 |
| ③ | 휴직자/휴직예정자 | 휴직시작일 기준 | {기준년도}.03.20 이전 휴직시작 → 제외 (연중 복직시 지급) |
| ④ | 직원구분 제외 | 직원구분 | 고문, 자문, 개인사업자, 아르바이트, 도급 → 제외 |
| ⑤ | 출산→육아 연차소진 | 실근무 여부 | 출산→육아 연차소진 중 실근무 없음 → 제외 |

### 세부 기준
- 퇴사예정자: {기준년도}.03.20 이후 퇴사자는 지급
- 휴직예정자: {기준년도}.03.21 이후 휴직시작은 지급
- 출산→육아: 연차소진 기간 실근무 있으면 지급, 없으면 제외
- 지급 직원구분: 정규직, 임원, 계약직, 전문계약직, 파견사원

### 업로드 템플릿 컬럼

| 컬럼명 | 필수 | 형식 | 설명 |
|--------|------|------|------|
| No | O | 숫자 | 순번 |
| 고유사번 | O | 텍스트 | 고유 식별자 |
| 사번 | O | 텍스트 | 사번 |
| 성명 | O | 텍스트 | 이름 |
| 파견업체 | X | 텍스트 | 파견사원인 경우 |
| 본/지점 | O | 텍스트 | 본점/지점 |
| 그룹입사일 | O | YYYY.MM.DD | 입사일 |
| 계약종료일 | X | YYYY.MM.DD | 계약직인 경우 |
| 직원구분 | O | 텍스트 | 정규직/임원/계약직/전문계약직/파견사원/도급/고문/자문/개인사업자/아르바이트 |
| 퇴사일 | X | YYYY.MM.DD | 퇴사자/퇴사예정자 |
| 휴직시작일 | X | YYYY.MM.DD | 휴직자 |
| 휴직종료일 | X | YYYY.MM.DD | 휴직자 |
| 휴직사유 | X | 텍스트 | 출산휴가, 육아휴직, 가족돌봄, 무급, 병가 등 |
| 출산→육아 연차소진(실근무無) | X | O/X 또는 빈값 | O이면 실근무 없음 → 제외 |

## 3. 구현 스펙

### 기술 스택
- 단일 HTML 파일 (index.html)
- Tailwind CSS (CDN)
- SheetJS/xlsx (CDN) - Excel 읽기/쓰기
- Vanilla JavaScript - 판정 로직

### 폴더 구조
```
/황금노트
  ├── index.html          # 메인 웹앱
  ├── blueprint-황금노트.md  # 설계서
  └── 피복비_판정_계산기.xlsx  # 원본 참고 파일
```

### 워크플로우
```
[기준년도 설정] → [템플릿 다운로드] → [데이터 입력] → [파일 업로드]
    → [5가지 규칙 자동 판정] → [결과 화면 표시] → [결과 Excel 다운로드]
```

### 화면 구성
1. **사이드바**: 메뉴 목록 (확장 가능)
2. **메인 영역**:
   - 기준년도 선택
   - 템플릿 다운로드 버튼
   - 파일 업로드 영역
   - 판정 결과 테이블 (지급/제외 구분, 제외사유 표시)
   - 통계 요약 (전체/지급/제외 인원)
   - 결과 다운로드 버튼
